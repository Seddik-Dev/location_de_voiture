<h2>🌍 Overview</h2>

<p>
Atlas Drive is a modern and premium car rental website designed specifically for the Moroccan market.
It serves as a professional demo template to present to car rental business owners.
</p>

<p><strong>Target Cities:</strong></p>
<ul>
  <li>Casablanca</li>
  <li>Rabat</li>
  <li>Marrakech</li>
  <li>Tanger</li>
  <li>Agadir</li>
</ul>
